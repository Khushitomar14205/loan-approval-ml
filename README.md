# Loan Approval Prediction System

This project predicts whether a loan should be approved based on applicant details using Machine Learning.

## Features
- Data preprocessing and cleaning
- Feature engineering (TotalIncome, LoanAmount_per_Income)
- Logistic Regression & Random Forest models
- Class imbalance handling
- Model evaluation using accuracy, precision, recall

## Results
- Accuracy: ~78%
- Improved rejection detection using class balancing

## Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy

## How to Run
1. Install requirements
2. Run notebook
3. Use model for prediction
